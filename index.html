安装说明

Windows

1、下载 color2gray-windows-v0.0.1 至本地

2、右键下载好的文件，选择「解压到 color2gray-windows-v0.0.1」

3、进入解压后的 color2gray-windows-v0.0.1 文件夹

4、双击 windows-installer.bat

5、会弹出终端，如果最后面出现提示 install success, please restart photoshop 则说明安装成功，重启 PS 即可

PS: 如果没有出现成功提示，则需要手动安装

找到插件目录的情况下，将下载好的 color2gray.zxp 文件改名为 color2gray.zip，再拷贝到插件目录后解压至 color2gray 文件夹，重启 PS 即可。
