Color2Gray - 更直观的 photoshop 明度判断插件

在画色彩的时候，如果想要画面丰富度高一些，或者多一些细节，可以通过「藏色」来实现。即将某个颜色用相同明度的颜色给替换一部分。

但是选择「明度相同的颜色」是比较困难的，该插件就是用来解决该问题的。在选择颜色后，能直观给出所选颜色的明度值，再也不用靠猜测了。

安装说明

插件本体是 color2gray.zxp 这个文件，只需要将其解压到 photoshop 的插件目录即可。

一般来说 Windows 插件目录在 C:\Program Files (x86)\Common Files\Adobe\CEP\extensions；macOS 插件目录位于 ~/Library/Application Support/Adobe/CEP/extensions

随插件带有一个命令行，在理想情况下双击即可帮助完成拷贝工作。

Windows

1、下载 color2gray-windows-v0.0.1 至本地

2、右键下载好的文件，选择「解压到 color2gray-windows-v0.0.1」

3、进入解压后的 color2gray-windows-v0.0.1 文件夹

4、双击 windows-installer.bat

5、会弹出终端，如果最后面出现提示 install success, please restart photoshop 则说明安装成功，重启 PS 即可

6、压缩包和解压后的文件夹都可以删除

PS: 如果没有出现成功提示，则需要手动安装

找到插件目录的情况下，将下载好的 color2gray.zxp 文件改名为 color2gray.zip，再拷贝到插件目录后解压至 color2gray 文件夹，重启 PS 即可。

使用说明

重启 PS 后，可以在 窗口 > 拓展功能 找到 「brighness」，点击即可打开插件。

在颜色选择面板选择任意颜色，插件会显示出正确的明度值。如下图片说明，hsb(0, 100, 100) 的红色，对应的明度值是 30。
